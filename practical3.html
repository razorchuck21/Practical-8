Opening the Developer’s Console in Mozilla Firefox and Safari:

Mozilla Firefox: You can open the developer’s console in Firefox by using the keyboard shortcut Ctrl + Shift + K for Windows/Linux or Cmd + Option + K for Mac123. Alternatively, you can access it from the menu bar: Menu -> Web Developer -> Web Console1.

Safari: To open the developer’s console in Safari, first ensure that the Develop menu is visible by going to Safari -> Preferences -> Advanced, and checking the Show Develop menu in menu bar checkbox. Then, you can open the console by selecting Develop -> Show JavaScript Console or using the keyboard shortcut Option-Command-C8.

Absolute vs Relative Links in HTML:

Absolute Links: These are links that contain the full URL, including the protocol (http/https) and domain name. They are generally preferable when you want to link to a resource that is hosted on a different domain

Relative Links: These are links that point to a file relative to the current page’s location. They are generally preferable when you want to link to a resource that is hosted on the same domain. This is because they are shorter and more portable

Image Formats Used in Web Development:

JPG: This is a lossy compression specification that is ideal for storing large photographic image files in small spaces. It’s not ideal for storing art files or images with sharp edges as they are often blurred or smeared out by anti-aliasing

GIF: This format is used for images that contain animations

PNG: This format is used for any image that needs transparency, or for images with text & objects with sharp contrast edges like logos.

WEBP: This is a modern image format developed by Google that provides superior lossless and lossy compression for images on the web SVG: Scalable Vector Graphics (SVG) is an XML-based markup language for describing two-dimensional based vector graphics. SVG images are scalable and do not lose any quality if they are zoomed or resized.
